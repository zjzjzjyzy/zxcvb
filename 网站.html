部署到 GitHub 全指南

  • 创建仓库：登录 GitHub，点击右上角「+」→「New repository」，命名为 username.github.io 或任意名称（例如 my-website）。
  • 上传文件：将本 HTML 文件重命名为 index.html（必须），也可以附带 CSS/图片等资源。直接拖拽上传到仓库。
  • 启用 Pages：进入仓库「Settings」→「Pages」→ 在「Branch」下选择 main 或 master 分支，根目录 / (root)，点击 Save。
  • 访问网站：等待1-2分钟，通过 https://你的用户名.github.io/仓库名/ 或 https://你的用户名.github.io/ 访问站点。
  • 自定义域名（可选）：在 Pages 设置中添加你的域名，并在 DNS 处配置 CNAME 记录。
💻 快速命令示例（本地提交）：
git init
git add index.html
git commit -m "✨ 首次部署简单网站"
git branch -M main
git remote add origin https://github.com/你的用户名/你的仓库.git
git push -u origin main
📌 小提示：如果仓库名为 用户名.github.io，则网站根地址为 https://用户名.github.io，否则为 https://用户名.github.io/仓库名。记得将 index.html 放在根目录！
